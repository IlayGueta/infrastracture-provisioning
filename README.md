# infrastracture-provisioning
Project description
DevOps Infrastructure Provisioning  &amp; Configuration Automation Project

Overview 
This is a rolling project that will evolve as new topics are learned. At this stage, the 
goal is to build the skeleton of an infrastructure provisioning tool. Future 
enhancements will integrate AWS and Terraform to create real resources. For now, 
the provisioning process is mocked to simulate infrastructure automation.
